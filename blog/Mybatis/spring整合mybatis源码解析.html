public class App {

		String resource = "mybatis‐config.xml";

​ FactoryBean接口的作用是在创建Bean的是通过getObject()方法返回的Bean，这也是为什么我写的spring整合mybatis的配置中通过SqlSessionFactoryBean的getObject()方法返回SqlSessionFactory，而有时候我们会看到配置的没有调用getObject()方法直接返回SqlSessionFactoryBean，就是因为SqlSessionFactoryBean实现了FactoryBean接口，就算我们不手动getObject()方法，Spring也会自动调用getObject()方法创建SqlSessionFactory的。
